<mode_behavior>
Lite mode: SQLite settles credits transactionally for one local gateway process.
Full mode: PostgreSQL settlement uses transaction plus row lock for concurrent gateway instances; Redis may coordinate auth/config hot state but not balance mutation.
Disabled mode: settlement is unavailable and must fail closed or report disabled.
</mode_behavior>
