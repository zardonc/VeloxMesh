<threat_model>
STRIDE register:
- Spoofing: Write permission comes from injected Coordinator state, not request input. Disposition: mitigate in repository decorator.
- Tampering: Emit after SQLite success to avoid stream events for failed writes. Disposition: mitigate in decorator ordering.
- Repudiation: ChangeEvent includes operation metadata and timestamp for replay/audit. Disposition: mitigate in event schema.
- Information disclosure: Stream events must not include secrets, raw prompts, vector embeddings, or ordinary-client topology details. Disposition: mitigate in event builder tests.
- Denial of service: Fenced writes fail fast without local queue growth. Disposition: mitigate per D-01.
</threat_model>
