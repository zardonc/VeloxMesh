<threat_model>
STRIDE register:
- Spoofing: Node IDs must come from trusted config or generated local identity; Redis owner checks prevent releasing another node's lease. Disposition: mitigate in RedisCoordinator.
- Tampering: Leadership writes use owner compare before renewal/release. Disposition: mitigate in RedisCoordinator.
- Information disclosure: Coordination details remain internal package state in this plan. Disposition: mitigate by no HTTP exposure.
- Denial of service: Redis errors make nodes non-writable instead of split-brain writable. Disposition: mitigate through IsWritable false on unknown state.
</threat_model>
