<threat_model>
STRIDE register:
- Information disclosure: offline exports and runtime artifacts must not contain raw prompts, auth material, provider secrets, payloads, or payload hashes. Disposition: schema allowlist tests and artifact publish tests.
- Tampering: scheduler runtime must be able to verify artifact integrity. Disposition: manifest checksum and parity check result.
- Repudiation: model artifacts need enough metadata to explain what was trained. Disposition: manifest includes feature schema, training window, and evaluation metrics.
</threat_model>
