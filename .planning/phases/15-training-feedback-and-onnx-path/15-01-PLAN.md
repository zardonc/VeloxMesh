- Denial of service: sample persistence must not make the data plane depend on training storage by default. Disposition: opt-in recorder and best-effort write behavior unless explicit strict feedback config is later introduced.
- Tampering: only gateway-created completion labels enter the sample repository. Disposition: no public data-plane write API for samples.
