<threat_model>
STRIDE register:
- Information disclosure: Migration and smoke logs must not print encrypted provider secrets, API keys, authorization headers, raw prompts, or DSNs with passwords. Disposition: redact report fields and use placeholders in docs.
- Tampering: Idempotent upserts must preserve source IDs and not invent records. Disposition: fixture tests and table-by-table report.
- Denial of service: Failed migration must stop instead of creating unknown partial state through skipped records. Disposition: stop-first failure behavior.
- Repudiation: Operators need an auditable completed/failed table report. Disposition: MigrationReport output.
</threat_model>
