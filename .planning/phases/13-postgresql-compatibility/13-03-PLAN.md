<threat_model>
STRIDE register:
- Information disclosure: Vector metadata must not include raw prompts or provider secrets. Disposition: metadata allowlist and tests.
- Tampering: Mixed embedding dimensions can corrupt search quality. Disposition: explicit dimension config and validation.
- Elevation of privilege: Cross-tenant cache hits can leak responses. Disposition: scope + model filtering in adapter/search mapping.
- Denial of service: pgvector unavailable should not break core forwarding when relational state is healthy. Disposition: degraded vector adapter path.
</threat_model>
