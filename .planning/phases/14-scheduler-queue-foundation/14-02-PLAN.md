<threat_model>
STRIDE register:
- Information disclosure: Redis queue members and task structs must not contain raw prompts, messages, auth headers, API keys, provider secrets, or original payloads. Disposition: queue stores task IDs and safe metadata only; tests grep forbidden fields.
- Denial of service: Queue growth can exhaust memory or Redis. Disposition: QueueGuard soft/hard limits and ErrQueueFull hard boundary.
- Tampering/replay: A queued task must not be executed after handler timeout or explicit cancellation. Disposition: ResultRegistry unregister and QueueBackend Remove support cancellation paths for 14-04.
</threat_model>
