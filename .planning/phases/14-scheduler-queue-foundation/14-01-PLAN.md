<threat_model>
STRIDE register:
- Information disclosure: scheduler.v1 and TaskFeature must not transmit raw prompts, messages, tool arguments, auth headers, API keys, provider secrets, or original request payloads. Disposition: contract allowlist and tests grep for forbidden field names.
- Denial of service: Scheduler endpoint latency or outage must not block Gateway startup or normal forwarding. Disposition: disabled default, 15ms timeout, no inline retry, breaker-open FIFO fallback.
- Spoofing/tampering: Untrusted text must not elevate priority. Disposition: proto/config expose only high/normal/low and later priority resolver owns trust policy.
</threat_model>
