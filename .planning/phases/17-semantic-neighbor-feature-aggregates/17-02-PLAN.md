## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Raw request in Gateway -> vector embedding | Raw request is used in memory only to create the search/index vector. |
| Vector metadata -> aggregate calculator | Only safe sample IDs and bounded scope fields are used to hydrate training samples. |
| Enricher -> Scheduler scorer | Only bounded TaskFeature aggregate fields cross into Scorer.Score. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
